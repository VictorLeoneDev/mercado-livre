# 🛒 Mercado Livre - E-Commerce Básico

## ⚡ COMO USAR (3 PASSOS)

1. **Abra a pasta** `public/`
2. **Clique duas vezes** em `index.html`
3. **Pronto!** O site abre no navegador ✅

## ✨ O QUE TEM

- 📸 4 produtos com imagens *(AGORA FUNCIONANDO!)*
- 🔍 Barra de busca
- 🛒 Botão adicionar ao carrinho
- 📱 Funciona em celular, tablet e desktop
- 🏷️ Descontos e avaliações
- 📂 Filtro por categorias

## 📝 EDITAR O PROJETO

### Editar CSS (estilos, cores, tamanhos)
1. Abra: `src/css/styles.css`
2. Faça suas mudanças
3. No terminal: `npm run build`
4. Refresh (F5) no navegador

### Editar JavaScript (cliques, alertas)
1. Abra: `src/js/app.js`
2. Faça suas mudanças
3. Refresh (F5) no navegador (sem compilar)

### Editar HTML (conteúdo, texto)
1. Abra: `public/index.html`
2. Faça suas mudanças
3. Refresh (F5) no navegador

- **HTML** - A estrutura da página
- **CSS** - As cores e estilos
- **JavaScript** - O que responde aos cliques
- **Node.js** - Apenas para compilar CSS (npm)
