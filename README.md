# Mercado Livre

Projeto de e-commerce com HTML, CSS e JavaScript.

## Como usar

1. Abra `public/index.html` no navegador
2. Teste a busca, adicione produtos ao carrinho e filtre por categoria

## Structure

```
public/         → Página HTML
src/            → CSS e JavaScript
imagem/         → Imagens de produtos
```

## Tecnologias

- HTML5
- CSS (Tailwind)
- JavaScript
